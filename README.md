# Aleks Pi

Personal voice assistant for Raspberry Pi.

## Goals

- Wake word detection
- Speech recognition
- Local command handling
- Smart home control (LEDs, Music etc.)
- Smart Answering of complex questions using LLMs

# Aleks

Offline voice assistant designed to run on Raspberry Pi.

## Run

source .venv/bin/activate
pip install requirements
python src/main.py
